# Remember the Eggs

This is a capstone project developed while a student at App Academy.  It is a clone of the popular to-do list app called Remember the Milk.  It involved coding a back-end utilizing an Express server framework with a Sequelize ORM. The front-end was built using React and makes use of Redux as a data store. No styling frameworks were used... just standard CSS.

Available for view at: [Remember the Eggs](https://davet-capstone.onrender.com/)
